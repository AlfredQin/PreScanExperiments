
% ================================================================================================
% Follow the next steps to create your own experiment with the Demo_Slalom
% ================================================================================================

Step 1.	Open the PreScan GUI and open the Demo_Slalom.pex file.

Step 2. click on the Invoke Simulation Run Mode Button and Matlab will be opened.

Step 3. In Matlab go to the directory where the experiment was saved. Then open the simulink file
	Demo_Slalom_cs.mdl file using Matlab.

Step 4. Start the Simulink model. (An initialisation function is implemented in order to load the model parameters)

Step 5. See the 3D visualisation viewer for the simulation result.
